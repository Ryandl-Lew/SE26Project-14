## 5. 技术选型评审记录

### 5.1 选型合理性评估

| 选型项 | 评估 | 依据 |
|--------|------|------|
| React 18 + Vite | ✅ 合理 | 成熟稳定、开发体验好、团队普遍熟悉 |
| Ant Design 5 | ✅ 合理 | 数据密集型应用首选、Table/Form强大 |
| Node.js + Express | ✅ 合理 | 前后端统一语言，降低上下文切换成本 |
| PostgreSQL + Prisma | ✅ 合理 | 关系型数据天然适合实验记录，Prisma降低SQL门槛 |
| ECharts | ✅ 合理 | 科学图表支持好，与Agent分析结果对接顺畅 |
| IndexedDB (Dexie.js) | ✅ 合理 | 满足草稿恢复需求，比LocalStorage容量大 |
