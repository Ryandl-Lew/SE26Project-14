# 界面原型设计 —— 生物实验助手

## 1. 设计原则

| 原则 | 说明 |
|------|------|
| 简洁优先 | 数据密集型应用，减少视觉干扰，让内容成为焦点 |
| 一致性 | 统一Ant Design设计语言，操作模式一致 |
| 响应式 | 适配PC端（≥1024px）、平板（768-1023px）、手机（<768px） |
| 渐进披露 | 复杂功能分层展示，常用操作触手可及 |
| 容错设计 | 危险操作二次确认，编辑状态自动保存提示 |

## 4. 关键界面说明

### 4.1 项目列表页（首页）

- 顶部搜索栏 + 新建项目按钮
- 卡片网格展示项目列表，每个卡片显示：项目名称、更新日期、数据记录数
- 空状态引导创建第一个项目
- 移动端切换为列表视图

### 4.2 数据记录页

- 左侧/顶部显示已定义的数据条目（动态表单）
- 每条条目：标签名 + 输入框 + 图片上传按钮
- 底部操作栏：保存 / 暂存草稿 / 取消
- 自动保存状态指示器

### 4.3 提醒设置页

- 时间线形式展示已设定的提醒
- 添加提醒弹窗：选择日期时间 + 重复规则 + 提前通知时间
- 可启用/禁用/删除已有提醒

### 4.4 数据分析页

- 顶部：选择数据条目 + 时间范围
- 中部：数学模型选择（可选）
- 图表区域：原始数据散点图 + 拟合曲线
- 底部：分析结果文字 + 指标表格
- "生成报告"按钮

## 5. 交互规范

| 交互 | 规范 |
|------|------|
| 加载状态 | Skeleton占位 / Spin加载中 |
| 空状态 | 插画 + 引导文案 + 操作按钮 |
| 错误状态 | Alert提示 + 重试按钮 |
| 表单验证 | 即时校验 + 提交时汇总错误 |
| 确认操作 | 删除等危险操作使用Popconfirm |
| 通知 | Message轻提示（成功/失败）、Notification系统通知（提醒） |

## 6. 响应式适配策略

| 断点 | 布局变化 |
|------|----------|
| ≥1200px | 完整侧边栏 + 多列内容 |
| 992-1199px | 折叠侧边栏 + 内容自适应 |
| 768-991px | 顶部导航 + 单列内容 |
| <768px | 汉堡菜单 + 全屏内容 + 底部操作栏 |
