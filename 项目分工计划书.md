# 项目分工计划书 —— 生物实验助手

## 1. 项目概况

| 属性 | 说明 |
|------|------|
| 项目名称 | 生物实验助手（BioExpAssistant） |
| 开发周期 | 10个工作日（2026-07-06 ~ 2026-07-17） |
| 团队人数 | 5人 |
| 技术栈 | React 18 + Ant Design 5 / Node.js + Express + PostgreSQL / Agent API |

## 2. 人员分工总览

| 编号 | 角色 | 姓名 | 职责范围 | 技术方向 |
|------|------|------|----------|----------|
| P1 | 前端负责人 | （待填） | 项目架构搭建、项目列表/详情/数据记录页面、公共组件、代码评审 | 前端 |
| P2 | 前端工程师 | （待填） | 提醒页面、数据历史页、数据分析页（图表）、公开数据页、响应式适配 | 前端 |
| P3 | 后端负责人 | （待填） | 数据库设计、项目/条目/数据CRUD API、文件上传、API文档 | 后端 |
| P4 | 后端工程师 | （待填） | 用户认证、提醒系统（定时+实时推送）、公开数据接口 | 后端 |
| P5 | Agent工程师 | （待填） | Agent分析服务、报告生成、数据查询Agent、草稿恢复（前端配合） | Agent |

## 3. 迭代任务分解（按2天粒度）

```
时间线:  D1-D2  │  D3-D4  │  D5-D6  │  D7-D8  │  D9-D10
阶段:   学习搭建 │ 基础功能 │ 核心功能 │ 核心功能 │ 联调交付
```

## 6. 风险与应对
